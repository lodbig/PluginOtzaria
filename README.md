## 🚀 התחלה מהירה

### דרישות קדם
- ✅ תוכנת **אוצריא** מותקנת
- ✅ דפדפן עדכני (Chrome, Firefox, Safari או Edge)
- ✅ מקום על הדיסק לשמירת קבצי התוספים

### התקנת תוסף בודד

#### שיטה 1: ייבוא ישיר מתיקיה
```
1. פתח אוצריא
2. עבור ל: הגדרות > תוספים > ייבוא תוסף חדש
3. בחר את תיקיית התוסף המבוקש
4. לחץ "ייבוא"
5. התוסף מופיע בטאבים
```

#### שיטה 2: העתקה ידנית
```
1. העתק את תיקיית התוסף למקום המוגדר בתוכנה
2. הפעל אוטומטית או טען מחדש את אוצריא
3. התוסף יטען אוטומטית
```

### התקנת כל התוספים

```powershell
# בצע זאת מתיקיית PluginOtzaria
foreach ($plugin in Get-ChildItem -Directory) {
    Write-Host "✓ ייבא את $($plugin.Name)"
    # העתק לתיקיית התוספים של אוצריא
}
```

## 📥 התקנה ידנית של תוסף

אם לא תעבוד הדרך האוטומטית, בצע התקנה ידנית:

### שלב 1: הורדה
1. בחר תוסף מהטבלה למעלה
2. לחץ על הלינק "⬇️ הורדה" 
3. קובץ ה-`.otzplugin` יתחיל להתהיל

### שלב 2: פתיחה בתוכנה
1. פתח את **אוצריא**
2. לך ל: **הגדרות** → **תוספים** → **ייבוא תוסף חדש**

### שלב 3: בחירת הקובץ
1. בחר את קובץ ה-`.otzplugin` שהורדת
2. לחץ **"פתח"** או **"ייבוא"**

### שלב 4: אישור
1. התוכנה תוודא את התוסף
2. לחץ **"אישור"** כדי להשלים ההתקנה
3. התוסף יופיע בטאבים הראשיים של אוצריא

### שלב 5: הפעלה
1. לחץ על הטאב של התוסף החדש
2. בעיקרון התוסף מוכן לשימוש מיידי

## 💡 טיפים מומלצים

### לשימוש יעיל בכל התוספים

1. **ארגון** - השתמש בקטגוריות ותיוגים בכל תוסף
2. **גיבוי שבועי** - ייצא נתונים בסוף כל שבוע
3. **תגובה** - השתמש בהערות ובתיאורים מפורטים
4. **סדר עבודה** - בנה שגרה קבועה עם התוספים
5. **שיתוף** - שתוף רשימות ודוחות עם חברים לימוד

### שילוב כל התוספים

```
דוגמה לשגרת לימוד יומית:

1. בתחילת היום - סימניה חכמה:
   הקפא את המקום שהפסקת אתמול

2. במהלך הלימוד - דפי מקורות + מראי מקומות:
   אסוף מקורות וציין מראי מקומות

3. בתום שיעור - סימניה חכמה:
   הקפא את המיקום החדש

4. בנק - שעון נוכחות:
   תעד את הנוכחות בקבוצה (אם בעל תפקיד)

5. בסוף חודש:
   בדוק דוחות במעשר כספים (אם רלוונטי)
```

   - תאר את השלבים להשחזור
   - צרף צילומי מסך או קודים שימושיים

3. **פתח Pull Request** 🔧
   ```bash
   # שלב 1: Fork את הריפו
   # שלב 2: צור Branch חדש
   git checkout -b feature/שם-התכונה
   
   # שלב 3: בצע את השינויים שלך
   # שלב 4: Commit לשינויים
   git commit -m "הוסף תכונה: תיאור"
   
   # שלב 5: Push ל-Branch
   git push origin feature/שם-התכונה
   
   # שלב 6: פתח Pull Request בגיטהאב
   ```

4. **שלח משוב** 📝
   - כתוב דוגמה שימוש
