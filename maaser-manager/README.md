# 💰 מעשר כספים

מערכת ניהול מעשרות וחומש לימודיות

## תיאור

מעשר כספים הוא תוסף מתקדם לניהול תרומות וחומש בהתאם להוראות התורה. התוסף מאפשר:

- 💸 ניהול מעשר כספים בקלות
- 🔐 הגנה בקוד גישה (PIN) אופציונלי
- 📊 ניהול תרומות קבועות
- 📈 סינון וביצוע דוחות לפי תקופה (יומי, חודשי, שנתי)
- 📁 ייצוא וייבוא מידע
- 🔔 התראות על תרומות שהגיעו
- 📅 שילוב עם לוח השנה

## מאפיינים

✨ **עיצוב מקצועי** - ממשק משתמש מודרני וידידותי  
🔒 **בטיחות** - הגנת PIN עם דוטים להסתרה  
📊 **דוחות מתקדמים** - צפייה בנתונים לפי תקופות שונות  
🎨 **עיצוב דינמי** - ישתנה עם כל ערכת צבעים  
📱 **ריספונסיבי** - עבודה מושלמת בכל גדלי מסכים  
⏰ **התראות** - קבל התראות על חשוב אירועים

## הוראות התקנה

### שלב 1: העתקת קבצי התוסף
1. הוצא את קבצי התוסף לתיקיה בחירתך
2. ודא שיש קבצים: `index.html` ו-`manifest.json`

### שלב 2: ייבוא בתוכנה
1. פתח **אוצריא**
2. עבור ל**הגדרות > תוספים > ייבוא תוסף**
3. בחר בתיקיית `maaser-manager`

### שלב 3: הפעלה ראשונה
1. התוסף מופיע בטאב תוספים
2. לחץ על **מעשר כספים**
3. מלא את הנתונים ההתחלתיים שלך

## דוגמאות שימוש

### דוגמה 1: הוספת הכנסה חדשה

```
1. לחץ על "+ הוסף"
2. בחר סוג: "הכנסה"
3. הזן סכום: 1000 ₪
4. בחר קטגוריה: "משכנתא/דיור"
5. לחץ "שמור"
```

### דוגמה 2: הגדרת תרומה קבועה

```
1. עבור לטאב "תרומות קבועות"
2. לחץ "+ הוסף תרומה קבועה"
3. הגדר:
   - שם: "חסד שלום"
   - סכום: 250 ₪
   - תדירות: חודשית
4. לחץ "שמור"
```

### דוגמה 3: ציפיית דוח חודשי

```
1. בטאב "דוחות" בחר "סוג": חודשי
2. בחר חודש ושנה
3. צפה בסיכום:
   - סה"כ הכנסה
   - סה"כ תרומות
   - יתרה זמינה
4. יצא לפורמט PDF/Excel
```

### דוגמה 4: הגנה עם קוד גישה

```
1. עבור להגדרות
2. הפעל "הגנה עם PIN"
3. הגדר PIN בן 4 ספרות
4. בפעם הבאה שתכנס - תתבקש להקליד PIN
```

## פונקציות מתקדמות

### 📊 סוגי דוחות
- **דוח יומי** - התראות יומיות על עסקאות
- **דוח חודשי** - סיכום לפי חודש
- **דוח שנתי** - ביצוע תיקייה שנתיים

### 🔄 ייצוא וייבוא
```
דוח > אפשרויות > ייצוא > בחר פורמט:
- Excel (.xlsx)
- CSV (.csv)
- PDF (.pdf)
```

### 📌 קטגוריות ברירת מחדל
- משכנתא/דיור
- חינוך
- בריאות
- צדקה וחסד
- תרבות וספרות
- אחר

## טיפים שימושיים

💡 **טיפ 1**: הגדר PIN כדי להגן על נתוני הכנסתך  
💡 **טיפ 2**: בדוק דוחות ובצע חודשיות לניהול יעיל  
💡 **טיפ 3**: השתמש בקטגוריות לסיווג מדויק  
💡 **טיפ 4**: ייצא דוחות לגיבוי חודשיות  

## שאלות נפוצות

**Q: איך אני משנה PIN?**
A: הגדרות > אמצעי אבטחה > שנה PIN > קלד PIN ישן ואחר כך חדש

**Q: האם יכול להיות בעל שיא של תרומה?**
A: כן, המערכת תציע זכרון עבור תרומות קבועות

**Q: איך אני מחזיר עסקה בטעות?**
A: לך לדוח > לחץ על עסקה > בחר "בטל עסקה"
