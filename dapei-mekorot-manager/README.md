# 📚 מנהל דפי מקורות

תוסף לניהול ואיסוף דפי מקורות בתוכנת אוצריא

## תיאור

מנהל דפי מקורות הוא כלי חיוני לכל הלומדים המקצועיים. התוסף מאפשר לך ליצור, לנהל ולאחסן רשימות מסודרות של דפי מקורות עבור כל הנושאים שלך. תוכל להשתמש בו כדי:

- 📝 ליצור דפי מקורות חדשים
- 🏷️ לארגן מקורות לפי קטגוריות
- 🔍 לחפש ולמצוא מקורות בקלות
- 💾 לשמור את כל המידע שלך באופן מאובטח
- 🔗 לפתוח מקורות ישירות מתוך אוצריא

## מאפיינים

- ✅ ממשק משתמש נקי וידידותי
- ✅ אחסון מוטבע לנתונים שלך
- ✅ יישום עם הנושא שנבחר בתוכנה
- ✅ גישה מהירה למקורות לימודיים
- ✅ ניהול מונע לרשימות מקורות

## התקנה

### שלב 1: יבוא התוסף
1. פתח את תוכנת **אוצריא**
2. לך לעמוד **התוספים** (Plugins)
3. בחר **ייבוא תוסף חדש** (Import Plugin)
4. בחר בקובץ זה או העתק את תיקיית התוסף

### שלב 2: הפעלת התוסף
1. לאחר ההתקנה, תראה כרטיסית חדשה בשם "דפי מקורות"
2. לחץ על הכרטיסית להפעלה

### שלב 3: התחלה
1. לחץ על כפתור **הוסף** (➕) ליצירת דף מקורות חדש
2. מלא את פרטי המקור ולחץ **שמור**

## דוגמאות שימוש

### דוגמה 1: יצירת דף מקורות לשיעור סוכות

```
שם: מקורות סוכות
תיאור: מקורות ליום טוב סוכות
מקורות:
- פרק כא פסוק לד (ויקרא)
- פרק טז פסוקים יג-יד (דברים)
- עמוד יח קו"א (סוכה)
```

### דוגמה 2: ארגון לפי נושא

```
נושא: הלכה
קטגוריה: שבת
יוצר: דף מקורות חדש עם כל הפסוקים בעלי קשר
```

### דוגמה 3: חיפוש מקורות

1. בחר בשדה "חפש"
2. הקלד מילה חיפוש (למשל: "טהרה")
3. תראה את כל המקורות הקשורים

## טיפים שימושיים

🎯 **עצה**: שמור את המקורות המשומשים ביותר בתחילת הרשימה לגישה מהירה

📌 **עצה**: הוסף תגיות לכל מקור כדי שתוכל לסנן בקלות

💡 **עצה**: ייצא את רשימות המקורות שלך גיבוי

## שאלות נפוצות

**Q: איך אני מוסיף מקור מתוך אוצריא?**
A: בעת קריאה בספר, לחץ על "הוסף למקורות שלי" מהתפריט הימני

**Q: האם המידע שלי בטוח?**
A: כן! כל הנתונים שלך נשמרים באופן מקומי בהתקן שלך

**Q: איך אני מעביר את המקורות שלי למחשב אחר?**
A: השתמש בפונקציה "ייצוא" ואז "ייבוא" בהתקן החדש

## עזרה וחיבור

אם יש לך בעיות או הצעות לשיפור, צור קשר עם המפתח בדף האוצריא הרשמי
