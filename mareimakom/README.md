# 📍 מנהל מראי מקומות

ניהול רשימות ומראי מקומות עם הדגשה ישירה בטקסט

## תיאור

מנהל מראי מקומות הוא כלי משוכלל לניהול ותגיון של מראי מקומות (הערות וביאורים על טקסטים). התוסף משלב בין ניהול רשימות למתקדם ויכולות הדגשה ישירות בטקסט הספר. אתה יכול:

- 📝 ליצור ולנהל רשימות מראי מקומות
- 🎨 להדגיש קטעים בטקסט
- 🔗 לקשר בין מראי מקומות לטקסטים
- 📌 לציין הערות ודברי הבהרה
- 🔍 לחפש בקלות בין כל המראי מקומות
- 💾 לאחסן את הנתונים באופן מאובטח

## מאפיינים

- ✅ **ממשק משתמש אינטואיטיבי** - קל להבין ולהשתמש
- ✅ **הדגשה בטקסט** - הסתכל מראי מקומות ישירות בספר
- ✅ **ניהול מונע** - פקדי פעולה מהירים לכל עסקה
- ✅ **תיוג ותיוג** - סווג מראי מקומות לפי קטגוריות
- ✅ **חיפוש מתקדם** - מצא מראי מקומות בשניות
- ✅ **קישור לטקסט** - קשר לכל טקסט בספר

## הוראות התקנה

### דרך 1: ייבוא ישיר בתוכנה
1. פתח **אוצריא**
2. לך ל**הגדרות > תוספים**
3. בחר **ייבוא תוסף חדש**
4. בחר בתיקיית `mareimakom`

### דרך 2: העתקה ידנית
1. העתק את תיקיית התוסף למקום המוגדר
2. הפעל אוצריא
3. התוסף יטען אוטומטית

## דוגמאות שימוש

### דוגמה 1: יצירת רשימת מראי מקומות חדשה

```
1. פתח את התוסף "מראי מקומות"
2. לחץ על "+ רשימה חדשה"
3. הזן שם: "דברים על חגים"
4. בחר קטגוריה: "חגים"
5. לחץ "צור"
```

### דוגמה 2: הוספת מראה מקום לרשימה

```
1. פתח רשימה קיימת
2. לחץ "+ הוסף מראה"
3. מלא את הפרטים:
   - מקור: (ספר ח"פ עמוד)
   - תיאור: (הערתך)
   - קטגוריה משנית (אופציונלי)
4. לחץ "שמור"

דוגמה:
   מקור: בראשית יא כז
   תיאור: כאן הולדת אברהם
   קטגוריה: אבות
```

### דוגמה 3: הדגשת טקסט בספר

```
1. בעת קריאה בספר, בחר קטע טקסט
2. לחץ "הדגש כמראה מקום"
3. בחר רשימה: "דברים על חגים"
4. בחר צבע הדגשה (אדום/כחול/ירוק)
5. הטקסט יודגש ויתווסף למראי המקומות
```

### דוגמה 4: חיפוש ומציאה

```
חיפוש בתוך רשימה:
1. לחץ בשדה "חפש" 
2. הקלד מילה חיפוש (למשל: "אברהם")
3. כל המראים המכילים את המילה יופיעו

חיפוש גלובלי:
1. לחץ על "חיפוש מתקדם"
2. בחר רשימות להחזקה
3. הקלד קריטריון חיפוש
```

### דוגמה 5: ניהול קטגוריות

```
1. עבור להגדרות > קטגוריות
2. לחץ "+ קטגוריה חדשה"
3. הוסף קטגוריה:
   - שם: "אבות"
   - צבע: כחול
   - סמל: 📖
4. לחץ "שמור"

קטגוריות רגילות:
- אבות
- חגים
- מצוות
- הלכה
- אגדה
```

## פונקציות מתקדמות

```
תיוג אוטומטי:
- מראים על אבות: #אבות
- מראים על ספרים: #ספרים
- מראים על פרקים: #פרק
```

### 🔗 קישור בין מראים
```
בעת יצירת מראה, קבע קישורים:
- מראים קשורים (אחרים המעתיקים את הנושא)
- הערות חוצות (עבור להערה אחרת)
```

### 📊 דוח ברשימה
```
1. בחר רשימה
2. לחץ "דוח"
3. צפה ב:
   - מספר מראים
   - קטגוריות בשימוש
   - מקורות שכיחים ביותר
4. יצא כ-PDF או Excel
```

## טיפים שימושיים

💡 **טיפ 1**: השתמש בצבעים הדגשה שונים לקטגוריות שונות - זה מקל על ההבחנה  
💡 **טיפ 2**: בדוק דוח רשימה חודשית כדי לראות את קדם ההבנה  
💡 **טיפ 3**: התאמת קטגוריות לצרכים שלך - אלו רק הצעות  
💡 **טיפ 4**: שתוף רשימות עם חברים במהלך לימוד קבוצתי  

## שאלות נפוצות

**Q: איך אני מעביר מראי מקומות מרשימה אחת לאחרת?**
A: בחר מראים > אפשרויות > העתק לרשימה > בחר רשימת יעד

**Q: האם אני יכול לשנות צבע הדגשה לאחר שהדגשתי?**
A: כן, לחץ על ההדגשה בטקסט > בחר צבע חדש

**Q: האם נתוני שלי בטוחים?**
A: כן, הנתונים שלך מאוחסנים בהתקן שלך בלבד

**Q: איך אני שותף רשימה עם אחרים?**
A: רשימה > ייצא > השתמש בקובץ ZIP > שלח לאחר > אחר יכול לייבוא

## הוראות יצוא וייבוא

### ייצוא רשימה
```
1. בחר רשימה
2. לחץ על שלוש הנקודות (...)
3. בחר "ייצא"
4. בחר פורמט: JSON/CSV/PDF
5. בחר מקום שמירה
```

### ייבוא רשימה
```
1. לחץ "ייבוא רשימה"
2. בחר קובץ
3. בחר רשימה שלייבוא
4. לחץ "ייבוא"
```
