# 🔖 סימניה חכמה - הקפאת לימוד

מקפיא את הלימוד שלך בלחיצה ימנית ושומר כל מה שחשוב לך

## תיאור

סימניה חכמה היא תוסף חינוכי מתקדם המאפשר לך להקפיא את מצב הלימוד שלך בדיוק כפי שהוא. בלחיצה ימנית בודדת, התוסף:

- 🔒 **מקפיא** את הטקסט שקורא כרגע
- 💭 **שומר** את המחשבות והערות שלך
- 📍 **זוכר** את המיקום המדויק בטקסט
- 🖊️ **שומר** את סימוני ההדגשה שלך
- ⏸️ **מחזיר** אותך בדיוק לאותה הנקודה כשתחזור

## מאפיינים

✅ **הקפאה מיידית** - סימניה בלחיצה ימנית יחידה  
✅ **שמירת מחשבות** - הערות ומחשבות כל הזמן  
✅ **זכרון מדויק** - חזור בדיוק לנקודה שהפסקת  
✅ **שמירה אוטומטית** - הכל משמר בלי צורך בפעולה  
✅ **עיצוב דינמי** - משתנה עם כל ערכת צבעים בתוכנה  
✅ **ממשק פשוט** - קל להבין ולהשתמש  
✅ **אנטגרציה מלאה** - עובד בצורה חלקה עם אוצריא

## הוראות התקנה

### שלב 1: בחר את מקום התוסף
1. הוצא את תיקיית `smart_bookmark` למקום בחירתך
2. ודא שקיימים `index.html` ו-`manifest.json`

### שלב 2: ייבוא בתוכנה
1. פתח **אוצריא**
2. עבור ל**הגדרות > תוספים > ייבוא תוסף חדש**
3. בחר בתיקיית `smart_bookmark`
4. לחץ "ייבוא"

### שלב 3: הפעלה
1. התוסף יופיע בתפריט ימני בעת קריאה
2. לא צריך עוד הגדרה - תוכנית פעולה מיד

## דוגמאות שימוש

### דוגמה 1: הקפאה בסיסית

```
בעת קריאה:
1. בחר קטע טקסט שאתה קורא
2. לחץ-ימני בתוך הטקסט
3. בחר "הקפא לימוד זה"
4. התוסף שומר:
   - המקור שקראת (ספר, פרק, שורה)
   - הטקסט המדויק שקראת
   - הזמן שבו הקפאת
```

### דוגמה 2: הוספת הערות תוך ההקפאה

```
כאשר אתה בתוך הקפאה:
1. לחץ "הוסף הערה"
2. הקלד את החשבונות שלך:
   "צריך לחזור לתקצור הקודם על הרבה"
3. הערה שמורה אוטומטית
4. לחץ בחזרה לעריכה בדוח
```

### דוגמה 3: חזרה להקפאה

```
כאשר תרצה להמשיך:
1. פתח את סימניות הלימוד שלי
2. בחר "הקפאות שלי"
3. לחץ על ההקפאה שברצונך
4. התוסף יפתח בדיוק לאותו המקום:
   - אותו הספר
   - אותו הטקסט
   - אותה הנקודה בדיוק

כשתחזור, תוכל:
- לקרוא את ההערות שהוספת
- להמשיך מהמקום שהפסקת
- לרחוב עוד הערות
```

### דוגמה 4: ניהול הקפאות מרובות

```
עם זמן, תצטבר לך מספר הקפאות:

בממשק הסימניות:
1. עבור לטאב "הקפאות שלי"
2. לחץ "ערוך הקפאות"
3. תוכל:
   - לשנות שם של הקפאה
   - לערוך הערות
   - למחוק הקפאות שלא צריך
   - לחפש בין הקפאות
```

### דוגמה 5: סדרנות וקיבוץ

```
ארגון הקפאות לפי נושא:
1. בממשק, לחץ "צור סדרה חדשה"
2. השם: "מסכת ברכות - שיעור 3"
3. בחר הקפאות להוספה לסדרה
4. כעת תוכל לנווט בקלות בין הקפאות הקשורות

דוגמה:
סדרה: "תנ"ך - תהלים"
  - הקפאה 1: פרק א'
  - הקפאה 2: פרק ב'
  - הקפאה 3: פרק ג'
```

### דוגמה 6: ייצוא וגיבוי הקפאות

```
גיבוי ההקפאות שלך:
1. לחץ על שלוש הנקודות (הגדרות)
2. בחר "ייצא הקפאות"
3. בחר פורמט: JSON/ZIP
4. לחץ "ייצא"
5. שמור בקובץ בטוח

טעינה מחדש:
1. לחץ "ייבוא הקפאות"
2. בחר קובץ קודם
3. לחץ "ייבוא"
```

## פונקציות מתקדמות

### 🎨 סימוני הדגשה
```
הקפאות שומרות גם:
- טקסט המודגש בירוק
- טקסט המודגש בצהוב
- הערות אישיות על המודגש
```

### 📌 קטגוריות מהירה
```
בעת הקפאה, בחר קטגוריה:
- לימוד רגיל
- דברים להדגשה
- שאלות שלי
- תשובות מעניינות
```

### 🔍 חיפוש וסינון
```
חפש בהקפאות שלך:
1. לחץ "חיפוש"
2. הקלד מילה חיפוש
3. תראה כל הקפאות הקשורות

סינון לפי:
- תאריך
- ספר/מקור
- קטגוריה
- רמת חשיבות
```

### 📊 דוחות ותיקיות

```
בממשק "דוח":
- כמה הקפאות ביצעת
- כמה שעות לימדת
- מה הנושאים המעניינים ביותר
- תרומה סטטיסטיקה
```

## שאלות נפוצות

**Q: האם קיימת הקפאה מגבלת?**
A: לא, תוכל להקפיא כל כמה שתרצה הקפאות

**Q: האם ההקפאות משמורות בכל מקום?**
A: כן, כל ההקפאות משמורות בהתקן שלך

**Q: האם אוכל לשתוף הקפאות?**
A: כן, ייצא הקפאה > שלח לאחר > אחר יוכל לייבוא

**Q: מה אם מחקתי הקפאה בטעות?**
A: לך ל"סל אחזקה" - הקפאות נשמרות שם 30 ימים

**Q: האם אוכל לערוך הערה אחרי שהקפאתי?**
A: כן, בכל עת לחץ על ההקפאה > בחר "ערוך"

**Q: איך אני משנה את הקטגוריה של הקפאה?**
A: בחר הקפאה > לחץ "שנה קטגוריה" > בחר קטגוריה חדשה

```

2. הגדר:
   - שם: "תלמידות שבוע זה"
   - תיאור: "כל השיעורים של השבוע"
   - ימי הוצגה: שני-שישי
3. הוסף הקפאות לסדרה
4. סדר הקפאות כרצונך
5. לחץ "התחל לימוד"
```
