# ⏰ שעון נוכחות

מערכת רישום נוכחות לתלמידים ומשתתפים בחינוך

## תיאור

שעון נוכחות הוא מערכת ניהול נוכחות מתקדמת המעוצבת במיוחד עבור מסגרות לימוד, כיתות וקבוצות לימוד. התוסף מאפשר:

- 📍 רישום כניסה ויציאה של משתתפים
- 👥 ניהול רשימות משתתפים
- 📊 ניהול וצפיית היסטוריה מלאה
- 📈 דוחות וסטטיסטיקות נוכחות
- 📤 ייצוא דוחות
- 🔔 התראות ודיוק זמן
- 📅 אינטגרציה עם לוח השנה

## מאפיינים

✨ **ממשק משוכלל** - עיצוב נקי וקל לשימוש  
⏱️ **רישום מדויק של זמן** - הזמן מתעדכן אוטומטית  
📊 **דוחות מתקדמים** - צפיית סטטיסטיקות שימושיות  
🎨 **עיצוב דינמי** - ישתנה לפי כל ערכת צבעים  
📱 **ריספונסיבי** - עבוד במכשירים שונים  
📅 **לוח שנה** - קראה מו-אוטומטית מלוח אוצריא  
🔔 **התראות חכמה** - הודעות על דברים חשובים

## הוראות התקנה

### שלב 1: הורדה והעתקה
1. הוצא את קבצי התוסף לתיקיה בחירתך
2. ודא שישנם קבצים: `index.html` ו-`manifest.json`

### שלב 2: ייבוא בתוכנה
1. פתח **אוצריא**
2. עבור ל**הגדרות > תוספים > ייבוא תוסף**
3. בחר בתיקיית `shmiz2`

### שלב 3: התקנה ראשונית
1. התוסף יופיע בטאב "תוספים"
2. לחץ על **שעון נוכחות**
3. בתפריט, בחר: **צור קבוצה חדשה**

## דוגמאות שימוש

### דוגמה 1: יצירת קבוצה חדשה

```
1. לחץ על "+ קבוצה חדשה"
2. מלא פרטים:
   - שם קבוצה: "כיתה יא"
   - מנהל: "מר שלום"
   - מקום: "חדר 205"
   - ימי לימוד: א', ב', ג', ד', ה'
3. לחץ "צור קבוצה"
```

### דוגמה 2: הוספת תלמידים

```
שיטה 1 - הוספה ידנית:
1. בחר קבוצה
2. לחץ "+ הוסף תלמיד"
3. מלא:
   - שם: "דוד כהן"
   - מס' תלמיד: 1234
   - כתובת דוא"ל: david@example.com
4. לחץ "שמור"

שיטה 2 - ייבוא מקובץ:
1. לחץ "ייבוא תלמידים"
2. בחר קובץ CSV/Excel
3. ודא שהתאמה נכונה
4. לחץ "ייבוא"
```

### דוגמה 3: רישום נוכחות יומי

```
בתחילת השיעור:
1. בחר תאריך היום (מעודכן אוטומטית)
2. בחר קבוצה
3. לחץ "כניסה"
4. סרוק/בחר את התלמידים הנוכחים
5. לחץ "אישור"

בסיום השיעור:
1. לחץ "יציאה"
2. בדוק את הנוכחים
3. רשום הערות אם נדרש
4. לחץ "שמור"
```

### דוגמה 4: צפיית היסטוריה

```
1. בחר קבוצה
2. לחץ "היסטוריה"
3. בחר תקופה: (היום / השבוע / החודש / בהתאם הבחירה)
4. בחר תלמיד (אופציונלי)
5. צפה ב:
   - תאריך כניסה
   - זמן כניסה
   - זמן יציאה
```

### דוגמה 5: דוחות סטטיסטיקה

```
דוח תלמיד בודד:
1. בחר קבוצה וזמן
2. לחץ על שם התלמיד
3. צפה ב:
   - אחוז נוכחות כולל
   - כמות ימי הנוכחות
   - כמות הימים הבנעדרים
   - אחוז יציאה עגומה

דוח קבוצה:
1. בחר קבוצה
2. לחץ "דוחות"
3. בחר "תקציר קבוצה"
4. צפה ב:
   - כמות ימי לימוד
   - סה"כ תלמידים
   - תלמידים עם 100% נוכחות
   - תלמידים עם נוכחות נמוכה
```

### דוגמה 6: ייצוא דוח

```
1. בחר קבוצה
2. לחץ על שלוש הנקודות (...)
3. בחר "ייצא דוח"
4. בחר תקופה וסוג דוח
5. בחר פורמט: PDF / Excel
6. לחץ "ייצא"
```

## פונקציות מתקדמות

### 🔔 התראות חכמות
```
הגדר התראות ל:
- תלמידים עם נוכחות נמוכה
- היעדר רצוף
- שינויים בנוכחות
```

### 📊 סוגי דוחות
- **דוח אישי** - לכל תלמיד בנפרד
- **דוח קבוצה** - סיכום הקבוצה כולה
- **דוח ניתוח** - ניתוח מגמות נוכחות
- **דוח השוואה** - השוואה בין תקופות

### ⚙️ הגדרות מתקדמות
```
הגדרות קבוצה:
- שעות התחלה וסיום
- מספר ימי לימוד בשבוע
- חופשות וימים מיוחדים
- רמות התראה על נוכחות נמוכה
```

## שאלות נפוצות

**Q: איך אני משנה שעה של כניסה/יציאה?**
A: היסטוריה > בחר יום > לחץ על הזמן > בחר זמן חדש

**Q: איך אני מחזיר עסקה?**
A: היסטוריה > בחר היום > לחץ על שלוש נקודות > בטל עסקה

**Q: איך אני מייבא תלמידים מגיליון Excel?**
A: לחץ "ייבוא תלמידים" > בחר קובץ > ודא תאימות > לחץ "ייבוא"

**Q: האם אוכל להוסיף תלמידים באמצע השנה?**
A: כן, בכל עת לחץ "+ הוסף תלמיד" בקבוצה

**Q: איך אני משתמש בקבוצות מרובות?**
A: צור קבוצה נפרדת עבור כל כיתה/קבוצה
